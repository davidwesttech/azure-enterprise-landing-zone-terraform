# Azure Enterprise Landing Zone with Terraform

## Overview

This project implements a production-style Azure Landing Zone using Terraform.

It establishes a governed Azure platform foundation including:

- Management group hierarchy
- Azure Policy guardrails
- RBAC access model
- Hub-and-spoke networking
- Centralized logging
- Terraform remote state
- CI/CD deployment pipeline

The goal is to demonstrate enterprise-grade Azure platform engineering practices.
